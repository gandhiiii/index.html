🔐 I'm the Admin — First-time system setup (click to expand)

Admin Setup

One-time setup. Works on Android, iPhone & Desktop. Live tracking via Firebase (free, no credit card).

Get free URL at console.firebase.google.com — see guide below.

📖 How to get your free Firebase URL (5 minutes, no terminal)
Step 1: Go to console.firebase.google.com (login with Google account)
Step 2: Click "Add project" → type any name e.g. hospital-fm → Continue → Continue → Create project
Step 3: Click "Build" in left menu → "Realtime Database"
Step 4: Click "Create Database" → choose any location → select "Start in test mode" → Enable
Step 5: You will see your database URL at the top — it looks like:
https://hospital-fm-12345-default-rtdb.firebaseio.com
Copy that URL and paste it in the field above

Step 6: Tap "🔌 Test Connection" — it should say ✅ Connected

⚠️ If live data is not showing on dashboard:
Go to Firebase Console → Realtime Database → Rules tab → Change rules to:
{"rules":{".read":true,".write":true}} Click Publish. Test mode rules expire after 30 days — this makes them permanent.

✅ Free forever. No credit card. No terminal. No code. Works on iPhone, Android & Desktop.

⚠️ Live data not showing? Fix Firebase Rules here
1. Go to console.firebase.google.com
2. Open your project → Realtime Database → Rules
3. Replace everything with:
{
  "rules": {
    ".read": true,
    ".write": true
  }
} 4. Click Publish → Done
5. Come back and tap Test & Save URL
